Flow will look for a configuration file with the following names:
- `flow.json`
- `flow.jsonc`
- `flow.yaml`
- `flow.yml`

## Options:

### `templatesPath`
Template directory

## Example:

```json title="flow.json"
{
    "templatesPath": "./templates"
}
```
